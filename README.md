# AVL Tree Visualizer

AVL Tree Visualizer is an interactive web-based application designed to visualize the AVL Tree data structure in real time. This project was developed as part of a Computer Graphics assignment and combines data structure algorithms with graphical visualization using HTML5 Canvas and JavaScript.

The application allows users to insert, delete, and reset nodes while automatically maintaining tree balance using AVL rotations. The visualization is presented with a modern and interactive user interface to help users better understand how AVL Trees work internally.

# AVL Tree Operations

## Insertion

Nodes are inserted following Binary Search Tree (BST) rules. After insertion, the AVL Tree automatically checks the balance factor and performs rotations when necessary.

## Deletion

Nodes can be removed dynamically while preserving AVL Tree balance through automatic rebalancing operations.

## AVL Rotations

The project implements all AVL balancing cases:

- Left Left Rotation (LL)  
- Right Right Rotation (RR)  
- Left Right Rotation (LR)  
- Right Left Rotation (RL)  

---

# Visualization

The AVL Tree is rendered dynamically using HTML5 Canvas. Each node is visualized as a circular object connected with smooth lines to represent parent-child relationships in the tree structure.
